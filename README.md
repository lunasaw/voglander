voglander

基于[sip-proxy](https://github.com/lunasaw/gb28181-proxy)等框架搭建的视频监控平台。支持模块化集群化部署，支持海康、大华、宇视、中维等主流监控设备接入，支持多种信令协议。包括但不限于GB28181、GT1078、ONVIF等协议。
